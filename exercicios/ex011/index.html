Colocando áudios

1° Tag


        <audio src="local do arquivo">

tag audio com tag source


        <audio>
            <source src="midia/guanacast-33.mp3" type="audio/mpeg">  
        </audio>

Tem mais type dependendo do formato do arquivo

  • <source src="local" type="audio/wav">
    •
  • <source src="local" type="audio/ogg">
    •
    • formato mais pesado EVITE usá-lo

Parâmetros da tag audio

  1. preload
    • auto
      • Considera que o site terminou de carregar quando terminar de carregar todo o áudio.
        Cuidado! Se o arquivo for grande ele irá demorar para carregar o áudio todo, logo, demorará pra considerar que o site carregou.
      • Vem por Padrão esta opção.
    • metadata
      • Carrega apenas o básico, como nome e duração.
    • none
      • Carregará Nada.
  2. autoplay

    3. O áudio é tocado automaticamente quando atualizada a página.

    <audio src="local" autoplay>

  3. controls

    4. Adiciona Controles de áudio.

    <audio src="local" controls>

  4. loop

    5. o som reiniciará após terminar, entrando em Loop.

    <audio src="local" loop>

Áudio tem Direito Autoral

As músicas também podem ter direitos autorais, assim como imagens e vídeos.

Ao utilizar um som com direitos autorais sem autorização pode acarretar em problemas para ti.

Utilizar sons sem direitos autorais é uma solução.

Sons sem direitos autorais

No Youtube tem um local cheio de sons que podem ser utilizados livremente, e dependendo será necessário apenas colocar os créditos do som.

Como encontrar:

  1. Vá no Youtube
  2. Entre no Youtube Studio
  3. Entre na seção Biblioteca de áudio

Pronto! Lá estará diversos sons disponíveis para download e utilizar no seu site sem preocupações!😀👍
