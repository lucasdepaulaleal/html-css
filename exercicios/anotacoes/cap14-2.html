Fontes no CSS

font-family

Ao abrir font-family, é onde escolhe a fonte, sendo possível colocar mais opções caso a primeira não funcione.
É chamada de safe combination.


            h1 {
                font-family: Arial, Helvetica, sans-serif;
            }

1° Usa a primeira fonte, se não utiliza a segunda, caso também não funcione ele vai para a terceira.
A 3° no caso vai procurar uma fonte sem serifa sozinho.

Tamanho, Peso, Estilo, shorthand

Tamanho

font-size
Medidas Absolutas:
cm, mm, in, px, pt, pc
Medidas Relativas:
em, ex, rem, vw, vh, %

Mais usados são em e px.

em

Tamanho do corpo da fonte. 1em = 1x; 2em = 2 vezes o tamanho da fonte...

peso

font-weight
lighther, normal, bold, bolder

estilo

font-style
ex. italic

shorthand (reduzido)

font: style weight size family;

Em um só é possível agrupar todos

font: italic bolder 3em Arial;

A ordem
font-style -> font-weight -> font-size -> font-family;

Fontes com Google Fonts

O Google Fonts é um reportório cheio de fontes para usar gratuitamente.

Como usar fontes externas?

Pode importar do google fonts, ele disponibiliza o código

Ele disponibilizará a Regra da fonte para colocar no CSS e a Declaração para usar a fonte.

Regra no CSS

Regras no CSS são indicadas pelo @

Como devem ser utilizá-las?

  • Deve estar nas Primeiras linhas do CSS
  • Não pode colocar dentro de um seletor

Exemplo:

@import url('https://fonts.googleapis.com/css2?family=Marck+Script&family=Work+Sans:ital,wght@0,100;0,200;0,300;0,400;0,500;0,600;0,700;0,800;0,900;1,100;1,200;1,300;1,400;1,500;1,600;1,700;1,800;1,900&display=swap');

Para utilizar a fonte:
font-family: 'Marck Script', cursive;
