Seletores no CSS

ID

ID é o identificador

Ele é introduzido dentro de uma tag. exemplo:
<h1 id="principal">

No CSS para selecioná-lo utiliza a #, exemplo:
h1#principal

Só é possível identificar 1 elemento

Class

Classificam um elemento.
Podem ser usados em mais de um elemento.

<h2 class="basico">

No CSS o seletor é .
Exemplo:
h2.basico

.

tag span

É apenas uma forma de selecionar uma parte do texto.
Uma forma de alterar algo dentro do texto no css, sem alterar todo o texto.

Pseudo-Classes em CSS

Representado pelo: :

                
                div > p {
                    display: none;
                }
            
                div:hover > p {
                    display: block;
                    color: white;
                    background-color: red;
                }
                
                div:hover {
                    color: red;
                }
