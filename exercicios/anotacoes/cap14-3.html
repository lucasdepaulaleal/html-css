Como utilizar fontes externas?

Baixe a fonte, no dafont é possível baixar.

Utilize a Regra @font-face e dá enter.

Vai ter o campo font-family, nele é onde se dar nome a fonte no seu código
src é para indicar o arquivo da fonte. entre parenteses é possível dar ctrl + espaço e procurar o arquivo da font

Declarações: format

É possível colocar a declaração format() na declaração src,
Nele indica o formato da fonte.
Os Tipos de format() são:

  • opentype (otf)
  • truetype (ttf)
  • embedded-opentype
  • truetype-aat (Apple Advanced Typography)
  • svg

Após isso já pode usar no seu seletor a fonte escolhida.
